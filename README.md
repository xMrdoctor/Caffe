# وب‌سایت کافه دنیس

وب‌سایت زیبا و کاربردی برای کافه دنیس با امکانات مدیریت منو و طراحی ریسپانسیو

## ویژگی‌ها

### 🎨 طراحی زیبا
- طراحی مدرن و جذاب با تم کافه
- رنگ‌بندی گرم و دوستانه
- استفاده از فونت فارسی Vazirmatn
- انیمیشن‌های نرم و جذاب
- طراحی کاملاً ریسپانسیو

### 📱 ریسپانسیو
- سازگار با تمام دستگاه‌ها
- بهینه‌سازی برای موبایل، تبلت و دسکتاپ
- منوی همبرگری برای موبایل
- تجربه کاربری یکسان در همه دستگاه‌ها

### 🍽️ مدیریت منو
- پنل مدیریت امن و کاربردی
- افزودن، ویرایش و حذف محصولات
- آپلود تصاویر محصولات
- دسته‌بندی محصولات (گرم، سرد، استایلیش، ویژه)
- فیلتر کردن محصولات
- نمایش قیمت‌ها به صورت فارسی

### 🔒 امنیت
- محافظت در برابر حملات CSRF
- Rate limiting برای جلوگیری از spam
- Hash کردن رمز عبور
- Validation کامل ورودی‌ها
- لاگ‌گیری از فعالیت‌ها
- محافظت از فایل‌های حساس

### 📊 آمار و گزارش
- نمایش آمار محصولات
- آمار دسته‌بندی‌ها
- لاگ فعالیت‌های مدیریت

## تکنولوژی‌های استفاده شده

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP 8.0+
- **Database**: MySQL 8.0+
- **Styling**: CSS Grid, Flexbox, Custom Properties
- **Security**: PDO, CSRF Protection, Rate Limiting
- **Icons**: Font Awesome 6
- **Fonts**: Vazirmatn (Google Fonts)

## نصب و راه‌اندازی

### 1. نصب محلی (WAMP)
راهنمای کامل در فایل `installation/WAMP_Installation_Guide.md`

### 2. انتشار روی هاست (cPanel)
راهنمای کامل در فایل `installation/cPanel_Deployment_Guide.md`

### 3. نصب سریع
```bash
# کلون پروژه
git clone [repository-url]

# کپی به مسیر وب سرور
cp -r cafe-dennis /var/www/html/

# تنظیم مجوزها
chmod 755 images/menu logs cache
chmod 644 config/database.php

# ایجاد پایگاه داده
mysql -u root -p -e "CREATE DATABASE cafe_dennis CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# ویرایش تنظیمات
nano config/database.php
```

## استفاده

### ورود به پنل مدیریت
- آدرس: `http://localhost/cafe-dennis/admin/`
- نام کاربری: `admin`
- رمز عبور: `CafeDennis2024!`

⚠️ **مهم**: حتماً رمز عبور پیشفرض را تغییر دهید!

### افزودن محصول
1. وارد پنل مدیریت شوید
2. فرم "افزودن محصول جدید" را پر کنید
3. تصویر محصول را آپلود کنید
4. دسته‌بندی مناسب را انتخاب کنید
5. روی "افزودن محصول" کلیک کنید

### مدیریت محصولات
- مشاهده لیست کامل محصولات
- حذف محصولات ناخواسته
- مشاهده آمار و گزارش‌ها
